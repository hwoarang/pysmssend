Pysmssend project is a program for sending sms over the internet using your ADSL/VOIP provider account. The program has both GUI and Command line interfaces so that it can fullfil any needs.
The providers currently supported are:

  • Otenet
  • Forthnet
  • Voipbuster
  • Voipdiscount
  • LowrateVoip
  • 12Voip
  • VoipbusterPro
  • Freevoip
  • WebcallDirect
  • Yahoo(beta)
  • Voipcheap
  • JustVoip

Please note that we havent tested all the providers. Fill a bug if you encounter any problem.

Also many development has been done to support Gentoo Ebuilds and use flag qt4. Gentoo users can compile it with or without qt4 flag in order to have gui or not (accordingly) interface. The Ebuild will be on Bugzilla soon.
Pkgbuild are also available on AUR for both i686 and amd64 architectures.
Packages for Ubuntu/Debian are also available.

We would like to inform you that we would be grateful to support more providers worldwide. In order to do this we will need your help as the Development Team is located in Greece thus we do not have accounts on foreign providers.

Hope you like the program and enjoy it. Please consult command line manual (pysmssend -h ) or README for more information.


If you want to help, you can contact me on hwoarang--at--silverarrow.org . Developers and Beta Testers are always welcome :)
